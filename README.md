# FunTnDBot

A Telegram bot for party-style games with random prompts:

- Truth
- Dare
- Would You Rather
- Roast Me
- Fate Prediction

The bot supports both private chats and group chats, and includes an inline button mode (`/play`) in private chat.

## Features

- Slash commands for each game mode
- Interactive inline keyboard in private chat (`/play`)

## Commands

- `/start` - Welcome message and quick usage
- `/help` - Help text based on chat type (private/group)
- `/play` - Open interactive buttons (private chat only)
- `/truth` - Random truth question
- `/dare` - Random dare challenge
- `/wyr` - Random "would you rather" question
- `/roast` - Random roast line
- `/fate` - Random fate prediction
